# Data Dictionary

| Column | Meaning |
|---|---|
| Name | Restaurant name |
| Address | Restaurant address |
| Province_City | Main province or city |
| Area | District or local area |
| Market_Scope | Whether the restaurant is inside the main target market |
| Avg_Rating | Average Google Maps rating |
| Total_Reviews | Total number of customer reviews |
| Bayesian_Score | Adjusted rating score considering review volume |
| Opportunity_Score | Score used to identify high-potential restaurants |
| Price_Tier | Estimated price segment |
| Has_Website | Whether the restaurant has a website |
| Has_Menu | Whether the restaurant has an online menu |
| Has_Delivery | Whether the restaurant supports delivery |
| Digital_Readiness_Score | Score based on phone, website, menu, and delivery availability |
